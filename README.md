# 🏦 Bank Management System

A full-stack **Bank Management System** built using **Spring Boot (Backend)** and **React (Frontend)**.
This application allows users to perform secure banking operations like registration, login, balance check, deposit, and withdrawal.

---

## 🚀 Features

* 🔐 User Registration & Login (JWT Authentication)
* 🚪 Secure Logout
* 💰 Check Account Balance
* ➕ Deposit Money
* ➖ Withdraw Money (with validation)
* 🔒 Spring Security Integration
* 🌐 RESTful API Architecture
* ⚡ Responsive UI with React

## AUTHENTICATION

* `POST /api/auth/register` → Register new user
* `POST /api/auth/login` → Login and get JWT token

### Banking Operations

* `GET /api/account/balance` → Check balance
* `POST /api/account/deposit` → Deposit money
* `POST /api/account/withdraw` → Withdraw money
* `POST /api/auth/logout` → Logout user
